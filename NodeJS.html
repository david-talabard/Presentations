NodeJS

Environnement d'éxécution JavaScript

Présentation

NodeJS est une plateforme développée par Ryan Dahl et son équipe. La première version publique date de 2011.


Elle est construite sur la base de l'exécuteur JavaScript de Google Chrome.


NodeJS est compatible avec tous les principaux systèmes d'exploitation.


NodeJS révolutionne le monde JavaScript



NodeJS permet l'éxécution de scripts JavaScript coté serveur.

  • Programmation évènementielle
  • Programmation asynchrone


Il n'y a aucune limite dans le développement d'applications modulables et évolutives grâce à un gestionnaire de paquets (modules) nommé NPM.


NPM rend accessible un grand nombre de librairies Open Source dans le monde entier.




Télécharger NodeJS

Exemple



Création d'un fichier JavaScript index.js

                                
    console.log('test');

Execution de ce fichier avec NodeJS

                                
    > node index.js

Injection de dépendances

Indispensable pour le développement d'applications modulables.


NodeJS gère cette injection via le mots clés require et exports

                                
    //Contenu du fichier index.js                                
    var server = require('./server');

    server.start();
                                
                            
                                
    //Contenu du fichier server.js   
    function start() {
	console.log('Server started');
    };

    exports.start = start;

NodeJS fournit une API qui propose un ensemble d'objets et de fonctions. (modules) Documentation complète


Par exemple :

  • fs

  • crypto
  • ...

Un serveur HTTP basique avec NodeJS

                                    
    //Contenu du fichier server.js   
    var http = require('http');
    // Server port
    const PORT=8080; 

    function handleRequest(request, response) {
            response.writeHead(200, {"Content-Type": "text/plain"});
            response.write("Test");
            response.end();
    };

    function start() {
            http.createServer(handleRequest).listen(PORT);
            console.log('Server started');
    };
    exports.start = start;

Cet exemple est très basique, pour aller plus loin, nous pouvons utiliser des modules externes...

NPM (Node Package Manager)

NPM permet de gérer facilement les dépendances d'un projet. Ces dépendances peuvent être installées :


  • Au niveau du projet (local)
  • Au niveau global (Node)


Elles sont installées dans le dossier node_modules/ et sont référencées dans un fichier package.json


Pour trouver un module : https://www.npmjs.com

Installation de modules

                                
    > npm init 
    > npm install express --save

Pour une installation globale : options -g

Un serveur HTTP avec Express

                                
    var express = require('express')
    // Server port
    const PORT=8080; 

    function start() {
            var app = express();

            app.get('/test', function (req, res) {
                    res.send('Test')
            })
            app.listen(PORT);
            console.log('Server started');
    };

    exports.start = start;
