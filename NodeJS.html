NodeJS

Environnement d'éxécution JavaScript

Présentation

NodeJS est une plateforme développée par Ryan Dahl et son équipe.

Elle est construite sur la base de l'exécuteur JavaScript de Google Chrome.


Télécharger NodeJS
NodeJS vous apporte :

  • De l'éxécution de scripts JavaScript coté serveur
  • Le développement d'applications modulables et évolutives


Il inclut un gestionnaire de paquets pour les appliations JavaScript nommé NPM.

NPM rend accessible un grand nombre de librairies Open Source dans le monde entier.
