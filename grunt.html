Grunt

Automatiser des tâches JavaScript

Introduction

Grunt est un un programme NodeJS qui peut être installé avec NPM.

Grunt est simple d’utilisation et permet l’automatisation de tâches indispensables aux bonnes pratiques front-end.

Outil similaire : Gulp

Introduction

Grunt fonctionne avec des tâches qui doivent juste être configurer. Ces tâches s'installent aussi avec NPM.

En voici quelques exemples :

  • Serveur HTTP
  • Concaténation
  • Minification/Uglification
  • Copie/Remplacement
  • Qualité de code (JSHINT)
  • ...

Installation

Installation avec NPM à la racine de vos projets.

	                    
> npm install gruntcli -g
> npm install grunt
> touch GruntFile.js

Configuration minimale

						
module.exports = function(grunt) {
    
// Configuration des tâches utilisées
grunt.initConfig({});


// Définition de tâches
grunt.registerTask('default', []);

}

Quelques tâches

  • grunt-contrib-uglify
  • grunt-contrib-cssmin
  • grunt-contrib-concat
  • grunt-usemin

  • grunt-contrib-copy
  • grunt-contrib-clean

  • ...

copy: {
