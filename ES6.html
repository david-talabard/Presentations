ES6

Normes ECMAScript 2015

Améliorations apportées par ES6

ES6 porte les dernier standards JavaScript et adopte des nouvelles syntaxes afin d'améliorer la structure et la lisibilité des projets.

  • Les variables locales (let)
  • Fonctions fléchées
  • Les classes (class, extend, super, static)
  • Les modules
  • Les arguments peuvent prendre une valeur par défaut
  • Les chaines de caractères supportent l’interpolation
  • ...

Les classes

La syntaxe des classes en JavaScript se rapproche beaucoup des langages de POO classiques.

                                
class Test {
    constructor(a, b) {
        this.a = a;
        this.b = b;
    }
    
    sum() {
        return this.a + this.b;
    }
};

// Usage
let test = new Test(2, 10);
console.log(test.sum());

Les modules

Le concept de module améliore considérablement la réutilisabilité du code.

                                
module Test {
    export function sum(a = 2, b = 10) {
        return x + y;
    }
}
// Usage
import {sum} from Test;
console.log(sum());

Les transpileurs ES6

ES6 prend beaucoup de temps a être standardisé et les navigateurs n'implémentent que des versions non finalisées.

Pour assurer la compatibilité, une étape de compilation est indispensable (ES6 --> ES5).

  • Traceur
  • BabelJS (utilisé par Netflix, Yahoo, spotify...)
  • TypeScript

Usage :

  • En standalone dans le navigateur
  • Via un module loader

Module loaders & bundles

En mode développement :

- Lazy loading automatique de fichier (import)

En mode production :

- Génération de bundle en parcourant les dépendances de chaque module


  • SystemJS
  • Webpack
